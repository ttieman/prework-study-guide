HTML

  • HTML is used to provide the structure and foundation to your website, and adds text, image, header, and list elements that are visbile to the user.
  • Three common element types that divide and describe pages are Meta-releated elements, content division elements, and semantic elements.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML is used in conjunction with other coding languages like CSS, and JavaScript to build a visually pleasing and functional website.

CSS

  • CSS deteremines how the webpage looks to the user, and determines font, layout, shadowing, color, and size of elements on the page.
  • There are three ways you can add CSS to your website, Inline CSS, Internal CSS, and External CSS.
  • Declerations contain two important components for determening the style of the elment selected. They contain a property and value, and these will effect the overal look of the element.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

GIT

  • git status: checks what branch we are currently on.
  • git checkout -b "branch-name":creates a new branch and switches to it.
  • git add -A: will commit all files in the working branch.
  • git commit -m "description of changes":commits the changes to the code.
  • git pull origin main: will pull the updated files to the main branch.
  • git push origin "branch that is pushing into the current selected branch": will create a pull request for approval to eventually be merged into the main branch.
